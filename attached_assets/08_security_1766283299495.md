# 08 — Seguridad

## Formula Injection (Excel)
Textos que empiezan con `= + - @` pueden ser interpretados como fórmula.
Mitigación:
- si es texto, prefijar `'` (apostrofe) para forzar texto.

Ver:
- `src/office_agent/renderers/excel.py` (`_maybe_escape_text`)
- `src/office_agent/quality/rules_excel.py` (warning)
