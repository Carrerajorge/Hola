# 07 — Deployment sin licencias

## Generación/descarga
Este repo genera `.docx/.xlsx`. El usuario puede abrir con LibreOffice/OnlyOffice/Google.

## Edición web sin Microsoft
Integra:
- OnlyOffice Document Server (self-host) o
- Collabora Online (WOPI)

Tu backend:
1) genera y guarda archivo (S3/MinIO)
2) el editor lo abre por URL y guarda de vuelta (callback)
