# 03 — Excel Rendering

Renderer: `src/office_agent/renderers/excel.py`

Incluye:
- Tablas (openpyxl.Table)
- Estilos de header
- Formatos numéricos
- Fórmulas por columna: `formulas: { "Total": "=C{row}*D{row}" }`
- Auto-fit heurístico de columnas
- Charts básicos (bar/line/pie)

Seguridad: mitigación opcional de **formula injection** (ver `docs/08_security.md`).
