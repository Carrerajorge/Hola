# 01 — Arquitectura

```
Prompt/Intención
  └─ (opcional) LLM → JSON (IR)
        └─ Repair loop si JSON inválido
  └─ Quality Gates (errors/warnings)
  └─ Renderer determinístico
        ├─ Excel (openpyxl)
        └─ Word  (python-docx)
  └─ Validator (reabre output)
  └─ Entrega (API/archivo)
```

## Por qué “determinístico”
Porque los formatos OOXML son estrictos; si el LLM “escribe” un .xlsx en texto, falla.
